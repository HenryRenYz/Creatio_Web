    安装Java

    记住这个Java，要考！

       如果你已经安装了 Java 且可以运行 Minecraft，则忽略这一章节。

       Java 是一种开源编程语言，Java 版 Minecraft 就是以 Java 为基础开发而成的，也就是Minecraft 的运行库。

            运行 Minecraft 需要 Java 运行环境 Java JRE（Java Runtime Environment）。高版本 Minecraft 需要 Java 8 来运行。Java 8 JRE 可在 Java.com 下载到，Windows 版也可以下载群文件中的 JRE-8u241-Windows64.exe


            我们还是建议你去 Java.com 下载，这是对开发者的鼓励和支持。

            在 Java 官网中选择与自己操作系统所对应的 Java ，Windows 版用户最好选择64位离线安装包（途中框出）。

       我们不建议你安装32位 Java ，因为32位应用程序限制 2G 内存会导致 Minecraft 卡顿，尤其是高版本异常的卡。

    下载Minecraft客户端

    做好准备！

            Minecraft与其他游戏一点明显的区别就是Minecraft的客户端与启动器是高度分离的，因此启动器多种多样，分为官方启动器和第三方启动器。

    我们建议你使用官方启动器来启动Minecraft客户端。

    在启动器中登录你的正版账号，下载Minecraft 1.15.2客户端就可以进入游戏了！

    如果游戏下载过于缓慢，或者连接不上服务器的话，也可以尝试使用第三方启动器进行下载

    MultiMC
    BMCL

    开始你的冒险

    添加服务器地址，开始服务器生活吧！

      在启动完成客户端后，我们进入 多人游戏 ，点击 添加服务器 按钮，进入界面后，在 服务器地址 一栏输入我们服务器的地址，优先选择使用 Play.Creatio.VIP ， 如果出现多次连接不上、频繁掉线、连接时间过长等问题请使用 MC.Creatio.VIP 。 服务器名称 一栏可以填写任何内容，这个只是客户端玩家方便识别服务器的标志。 群文件中预置的客户端已提前添加好了服务器，无需玩家再次手动添加。

    如果以上两条线路均有卡顿等问题，请查看玩家交流群的公告中的其他线路。

            如果在连接服务器的过程中出现了：长时间显示 登入中，长时间显示 加密通讯中，显示 身份验证正在停机维护、 认证服务器处于宕机状态中 字样，请尝试重启客户端和重连服务器，如果仍然不行请检查本地网络情况，尤其是防火墙设置。

            如果在连接服务器的过程中出现了：Connection timed out: no further information 、Unknown Host 或 无法解析主机名 字样，请尝试重启客户端和重连服务器，然后检查网络设置和防火墙，最后在QQ群中查看服务器是否开启。

            如果在连接服务器的过程中出现了 登入失败：无效的会话 字样，则说明你的正版登录已失效，或者你正在使用盗版进行登录，请确认自己登录的账号为正版账户，然后重启一次客户端，重新连接服务器。

            如果你在服务器里发现延迟过高（≥100ms），或出现频繁掉线的问题，请检查本地网络情况、运营商情况等。如果仍然无法解决请考虑挂加速器进行登录。
